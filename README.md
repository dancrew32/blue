# Blue

Experiments with `PyObjC` and the bluetooth libraries that it exposes for OSX.

## Examples

* `./venv/bin/python open_ui.py` to launch a bluetooth ui
* `./venv/bin/python list_devices.py` to list all recent bluetooth devices
